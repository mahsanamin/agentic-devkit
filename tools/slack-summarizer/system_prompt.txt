- The parent message (topic) is often just a question or escalation. The REAL substance is in the replies[].

- Summarize the OUTCOME and KEY DETAILS, not just the parent message.
- PAY SPECIAL ATTENTION TO THE LATEST REPLY — it defines the current state. If the latest reply is a resolution, say so. If it's a follow-up question, that's the pending action.

PERMALINK FORMAT — THIS IS CRITICAL, DO NOT GET THIS WRONG:
CORRECT: - @alice: deployed new service (<https://workspace.slack.com/archives/C09TR/p1741506795|view thread>)
WRONG:   - <@U123>: deployed new service
WRONG:   - @alice: deployed new service (view thread)
WRONG:   - @alice: deployed new service [link](url)
The permalink URL MUST be inside the <url|view thread> Slack link syntax. NEVER output bare text like "view thread" without wrapping it in <url|...>.
